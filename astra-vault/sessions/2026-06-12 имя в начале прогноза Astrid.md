# Имя в начале прогноза Astrid

## Решение

Первое предложение прогноза **обязательно** начинается с имени пользователя в **именительном падеже**:

> Марина, сегодня твоя внутренняя энергия…

Без склонений (`Марине`, `Аиду` — запрещено). Дальше только «ты», имя не повторяется.

## Изменения

- `src/astra/llm/prompts/astrid.py` — правило в system prompt
- При сбое LLM — retry в worker, без текстового fallback

## Прогоны gemma4:e2b
