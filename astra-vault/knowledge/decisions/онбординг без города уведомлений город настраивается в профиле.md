# Онбординг без города уведомлений — город настраивается в профиле

## Контекст

Шаг «Где ты сейчас живёшь?» удлинял регистрацию; timezone для рассылки 09:00 можно взять из места рождения до настройки.

## Решение

1. **Регистрация** заканчивается после выбора места рождения: `run_registration_phase` → commit → `run_greeting_phase`.
2. `notification_place_id` = NULL; `city`/`timezone` — из места рождения или дефолты.
3. **Профиль** → «🌍 Город для уведомлений» → тот же GeoNames picker (`ProfileStates.edit_notification_place_query`).

## Этапы

| Этап | Код |
|------|-----|
| Регистрация | `onboarding_service`, `places._complete_onboarding_after_birth_place` |
| Приветствие | `greeting_service`, `prediction_delivery_service` |
