# Имя в онбординге по умолчанию из Telegram, правка в Профиле

## Контекст

Отдельный шаг «Как тебя называть?» с подсказкой «отправь то же имя» — лишнее трение: имя уже есть в Telegram.

## Решение

- При «▶️ Начать» сохранять `display_name` из `first_name` / `username` / «друг» (`default_display_name`).
- Не выделять состояние FSM `name` в онбординге.
- Смена имени — «👤 Профиль» → «✏️ Имя» (`ProfileStates.edit_name`), как и раньше.

## Последствия

- Онбординг короче на один шаг.
- Профиль в БД создаётся по-прежнему в `finish_onboarding` с `display_name` из FSM.
