# Карточка профиля A2 — сокращение названий GeoNames

## Формат (Telegram HTML)

```
✨ Твой профиль

👤 {имя}

📅 ДД.ММ.ГГГГ
🕐 ЧЧ:ММ | подсказки если пусто

📍 {город, регион}

🌍 {город}
   ЧЧ:ММ · {timezone}

──────────────
🔥 Серия N дн.  ·  ⭐ M баллов
```

## Сокращение мест

- `shorten_place_display`: убрать «, Россия», «Республика » → «Козет, Адыгея».
- `shorten_city_label`: первая часть до запятой → «Краснодар».

Код: `src/astra/telegram/profile_text.py`.
