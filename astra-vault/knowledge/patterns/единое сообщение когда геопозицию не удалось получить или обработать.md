# Единое сообщение, когда геопозицию не удалось получить или обработать

## Паттерн

Константа + одна функция ответа:

```python
GEOLOCATION_FAILED_TEXT = "Не смог получить твою геопозицию, введи название в ручную"

async def reply_geolocation_failed(message: Message) -> None:
    await message.answer(
        GEOLOCATION_FAILED_TEXT,
        reply_markup=place_step_reply_keyboard(),
    )
```

Вызывать из всех веток сбоя: текст кнопки вместо location, пустые координаты, exception, пустой `find_nearest_places`, невалидный `web_app_data`.

## Зачем

Пользователь не читает длинные инструкции про 📎 и Web App; один понятный fallback → ручной ввод.

`src/astra/telegram/handlers/places.py`
