# Промпт Astrid — короткий прогноз 3–5 предложений

Связано: [[продукт Astra telegram предсказания RU аудитория]], [[стек Python 3.12 uv и FastAPI]]

## Роль

**Astrid** — профессиональный астролог-консультант (голос бота Astra). Не «дерзкий инсайт в 2 предложения», а персональный дневной прогноз.

## Код

- Промпт и постобработка: `src/astra/llm/prompts/astrid.py`
- Вызов Ollama: `src/astra/llm/ollama.py` (`temperature` 0.72, `num_predict` 450)
- Fallback без LLM: `src/astra/astro/templates.py`

## Входные данные для модели

В **user**-сообщение передаются:

| Поле | Источник |
|------|----------|
| дата / время / место рождения | `Profile` |
| текущая дата | `AstroContext.date` |
| Солнце, Луна, ASC | `NatalChartData` |
| положения планет | `NatalChartData.planets` |
| транзиты дня | `AstroContext` JSON (`transits[]`, `orb_deg`, `theme`) |

`build_user_message(ctx, profile, chart)` — сигнатура с 2026-06-02.

## Требования к тексту

- **3–5 предложений** — только основной блок (до «Совет дня»); `sanitize` обрезает лишнее.
- Темы вплетены в связный рассказ: эмоции, отношения, работа/финансы, энергия, действие дня.
- **Без рубрик** «Общая энергия дня», «Работа и деньги» и т.п.
- Обращение только на **«ты»** (не «вы»); мягкие формулировки («может ощущаться», «есть тенденция»).
- Запрет пустых фраз: «вас ждут перемены», «будьте внимательны» без контекста.
- **Только русский (кириллица)** — никаких иероглифов (CJK); `sanitize` вырезает CJK, если модель всё же вставила.

## Формат ответа (Telegram)

```
✨ Прогноз дня

[основной текст]

💡 Совет дня:
...

🔢 Число дня:
...

🎨 Цвет дня:
...
```

`sanitize_prediction_output` сохраняет структуру и эмодзи; при отсутствии заголовка добавляет `✨ Прогноз дня`.

## Точность профиля

- До 33%: в основном солнечный знак.
- от 66%: Луна и планеты (kerykeion).
- от 66% + время: ASC.

Модель не должна выдумывать Луну/ASC, если в данных «не рассчитан».

## История

- До 2026-06-02: короткий инсайт 1–2 предложения, без категорий.
- 2026-06-02: длинный прогноз 600–1000 символов.
- С 2026-06-02 (позже): снова короткий формат **3–5 предложений** + совет / число / цвет.
